Discord bot that gives Valorant info (player stats, match history, store info)  
  
Uses raimannma/ValorantAPI  
  
How to deploy:  
1. Create a tmux session.  
- tmux new -s DiscordBot  
  
2. Change to the Discord bot project directory.  
- cd ~/ValorantAPI/bot 

3. Start the bot.  
- python3 valbot3.py
